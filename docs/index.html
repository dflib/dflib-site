DFLib - Java DataFrame

DFLib is a lightweight pure Java implementation of a common DataFrame data structure. With DataFrame, you get data manipulation capabilities similar to SQL (filtering, joins, unions, and so on), only done as step-by-step in-memory transformations that are easy to compose and understand. DFLib comes with connectors for CSV, Excel, RDBMS, Avro, Parquet, JSON, etc. and can be easily adapted to other formats. It is open source and is distributed under the Apache License.
