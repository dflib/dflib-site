Home of open source Java data tools: a DataFrame library, charts and visualization, Jupyter kernel.

Why Java developers need a DataFrame

DFLib's DataFrame is a lightweight, pure Java data structure. It provides data manipulation capabilities similar to SQL - filtering, joins, unions, window functions, and more - only done as step-by-step in-memory transformations that are easy to compose and understand. Unlike "big data" DataFrames, DFLib does not require special infrastructure and can be used in any Java application. DFLib includes connectors for CSV, Excel, RDBMS, Avro, Parquet, JSON, etc. It is open source, distributed under the Apache License.
